# 분리고 앱 설치 및 실행 가이드

## 1단계 — Node.js 설치

1. https://nodejs.org 접속
2. **LTS** 버전 (왼쪽 녹색 버튼) 클릭해서 다운로드
3. 설치 파일 실행 → 계속 "다음" 클릭 (기본값으로 설치)
4. 설치 완료 후 PC 재시작

## 2단계 — 프로젝트 폴더 실행

1. 이 폴더(wastego)를 원하는 위치에 저장
2. **터미널(명령 프롬프트)** 열기
   - Windows: 시작 → "cmd" 검색 → 실행
   - Mac: Spotlight(⌘+Space) → "터미널" 검색 → 실행
3. 아래 명령어를 순서대로 붙여넣고 Enter:

```
cd 저장한경로/wastego
npm install
npm run dev
```

4. 브라우저에서 **http://localhost:5173** 접속
5. 앱이 뜨면 성공! 🎉

## 3단계 — Vercel로 배포 (인터넷에 공개)

1. https://github.com 가입
2. "New repository" → wastego 폴더 전체 업로드
3. https://vercel.com 접속 → GitHub 계정으로 로그인
4. "Add New Project" → wastego 저장소 선택 → Deploy
5. 완료되면 https://wastego-xxx.vercel.app 같은 주소 생성

## 4단계 — 모바일에 설치 (PWA)

배포된 주소를 스마트폰 브라우저로 접속:
- **iPhone**: Safari → 공유 버튼 → "홈 화면에 추가"
- **Android**: Chrome → 우상단 메뉴 → "앱 설치" 또는 "홈 화면에 추가"

아이콘이 생기고 앱처럼 실행됩니다!
